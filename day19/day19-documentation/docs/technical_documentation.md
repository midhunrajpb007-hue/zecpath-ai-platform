# ATS Engine – Technical Documentation

## 1. Overview
The ATS (Applicant Tracking System) engine is a Python-based system designed to automate resume screening, candidate scoring, and shortlisting. It extracts key information from resumes, matches candidates against job requirements, and generates ranked shortlists with explainable scores.
