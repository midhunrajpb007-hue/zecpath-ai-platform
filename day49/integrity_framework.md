# Integrity Detection Framework – Zecpath

## Objective
To detect cheating or external assistance during AI interviews using non‑invasive signals.

## Core Signals
- Tab switching frequency
- Screen focus loss
- External voice detection
- Gaze deviation

## System Architecture
Browser Activity Tracker → Focus Detection → Audio Analyzer → Behavioral AI → Signal Aggregator → Risk Engine → Flagging System → Recruiter Dashboard

## Privacy
- User consent required
- No raw audio/video storage
- Only aggregated metrics saved
