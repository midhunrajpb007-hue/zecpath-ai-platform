# Machine Test AI Framework – Zecpath

## Objective
Evaluate real‑world technical skills through automated coding, debugging, and design tasks.

## Components
- Task manager (selects task based on role & difficulty)
- Code editor + runtime sandbox (execute code safely)
- Test runner (runs hidden test cases)
- Evaluator (scores correctness, efficiency, quality, approach)
- Score aggregator

## Test Types
- Coding problems
- Debugging tasks
- File‑based tasks
- Mini system design

## Scoring Weight
- Correctness 40%
- Efficiency 25%
- Code quality 20%
- Approach 15%

## Time Factor
- Estimated time per task
- Partial credit if time exceeded (reasonable attempt)
